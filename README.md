#UAS_IRC_Bot
===========

The new Unallocated Space IRC Bot

Utilize UnalloBot3.conf to setup connection information using the template UnalloBot3.conf

Commands:
* !eightball or !8ball
* !site
* !address
* !status
	
More will be implemented as time goes on!
